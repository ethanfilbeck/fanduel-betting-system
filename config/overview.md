# FanDuel Sportsbook Betting System

## Purpose

Analyze sportsbook lines for inefficiencies and probabilistic edges.

## Scope

- Evaluate spreads, totals, moneylines
- Compare odds across markets
- Detect line movement patterns

## Inputs

- FanDuel odds data
- Game statistics
- Line movement history

## Outputs

- Value detection insights
- Risk assessment
- Comparative analysis

## Rules

- Research and analysis only
- No automated betting actions
